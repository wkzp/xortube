Как это работает:

Этот плеер загружает два видео (поддерживаются Streamable или прямые .mp4 ссылки) и выполняет попиксельную операцию XOR для создания визуального эффекта.

Важно: Для Streamable используйте прямые .mp4 ссылки (см. инструкции). Хостинг должен поддерживать CORS. Рекомендуемые сервисы: BunnyCDN, Amazon S3.

Для Streamable: откройте видео, нажмите правой кнопкой → Inspect → найдите <video> → скопируйте атрибут src.
